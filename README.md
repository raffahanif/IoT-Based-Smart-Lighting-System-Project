# IoT Based Smart Lighting System Project

Smart Automatic Lamp is a simple yet structured IoT-based project designed to control lighting automatically based on environmental light intensity.

This project uses a phototransistor to measure light intensity, allowing the system to automatically turn the lamp on or off depending on surrounding conditions. In addition, a manual control feature is also available using an IR remote, giving users flexibility to operate the lamp directly.

The system is fully integrated with the Blynk application, enabling users to monitor lamp status, view real-time light intensity values, and control the device remotely through a mobile interface.

The project is developed using Arduino IDE, focusing on practical IoT implementation, real-time sensor integration, and smart home automation concepts.

🚀 Features
1. Automatic lighting control based on light intensity (phototransistor)
2. Manual control using IR remote
3. Real-time monitoring via Blynk app
4. Lamp status and sensor value display
5. Simple and responsive IoT control system

🛠️ Tech Approach 
This project follows:
1. IoT-Based Development: Integration of sensors, microcontroller, and mobile app
2. Arduino IDE Implementation: Programming ESP/Arduino for hardware control
3. Real-Time Communication: Data monitoring and control via Blynk
4. Modular System Design: Separation of sensor input, control logic, and output

📌 Purpose 
The main goal of this project is to:
1. Implement smart home automation using IoT concepts
2. Combine automatic and manual control systems
3. Learn real-time sensor integration with microcontrollers
4. Build a practical and scalable smart lighting system
